Course Details


S7 Courses:

Artificial Intelligence

Artificial Intelligence covers topics such as intelligent agents, machine learning, neural networks, natural language processing, and robotics. It includes both theoretical foundations and practical applications in creating systems that can perform tasks typically requiring human intelligence.

Back to Top

Industrial Safety Engineering

Industrial Safety Engineering focuses on the principles and practices of ensuring safety in industrial environments. Topics include risk assessment, hazard identification, safety regulations, accident prevention, and the design of safety management systems.

Back to Top

PROGRAM ELECTIVE II

Machine Learning

Machine Learning covers algorithms and models for learning from data. Topics include supervised and unsupervised learning, reinforcement learning, neural networks, support vector machines, and decision trees. Practical applications in various fields are also explored.

Back to Top

Cloud Computing

Cloud Computing involves the delivery of computing services over the internet. Topics include cloud service models (IaaS, PaaS, SaaS), virtualization, cloud storage, cloud security, and the development and deployment of applications in cloud environments.

Back to Top

Security in Computing

Security in Computing addresses techniques and practices for protecting information systems. Topics include cryptography, network security, application security, cybersecurity policies, risk management, and incident response.

Back to Top

Model Based Software Development

Model Based Software Development focuses on the use of abstract models in the design and analysis of software systems. Topics include UML, model-driven architecture, model transformations, and verification and validation of models.

Back to Top

Web Programming

Web Programming covers the creation of dynamic and interactive web applications. Topics include HTML, CSS, JavaScript, server-side scripting, database integration, and web frameworks. Emphasis is placed on best practices for web development.

Back to Top

Natural Language Processing

Natural Language Processing involves the interaction between computers and human languages. Topics include text preprocessing, language modeling, parsing, sentiment analysis, machine translation, and speech recognition.

Back to Top


OPEN ELECTIVE

Introduction to Mobile Computing

Introduction to Mobile Computing covers the basics of mobile device technologies, mobile operating systems, and the development of mobile applications. Topics include mobile UI design, mobile app architecture, and mobile security.

Back to Top

Introduction to Deep Learning

Introduction to Deep Learning covers the fundamentals of deep neural networks. Topics include neural network architectures, training algorithms, convolutional networks, recurrent networks, and practical applications of deep learning.

Back to Top

Computer Graphics

Computer Graphics involves the creation and manipulation of visual content using computers. Topics include rendering techniques, modeling, animation, and the use of graphics APIs. Applications in gaming, simulations, and visual effects are explored.

Back to Top

Python for Engineers

Python for Engineers introduces Python programming for solving engineering problems. Topics include data structures, algorithms, data analysis, visualization, and scripting for automation in engineering tasks.

Back to Top

Object Oriented Concepts

Object Oriented Concepts cover the principles of object-oriented programming. Topics include classes and objects, inheritance, polymorphism, encapsulation, and abstraction. Emphasis is placed on designing and implementing object-oriented systems.

Back to Top


OTHERS

A Seminar involves student presentations and discussions on specialized topics. It provides a platform for students to develop their presentation and communication skills while deepening their understanding of the subject matter.

Back to Top

Project Phase I includes the initial stages of a project, focusing on research, planning, and proposal development. Students work on defining the problem, conducting literature reviews, and creating project plans.

Compiler Lab

Compiler Lab involves practical exercises in designing and implementing compilers. Topics include lexical analysis, syntax analysis, semantic analysis, code generation, and optimization techniques.

Back to Top

Minor Course

A Minor Course allows students to gain additional knowledge in a secondary field. Topics vary depending on the chosen minor, providing complementary skills and knowledge to the main degree program.

An Honours Course offers advanced studies and research opportunities in a specific subject. Topics are in-depth and research-oriented, providing specialized knowledge and insights beyond the standard curriculum.

S8 Courses:

PROGRAM ELECTIVE III

Distributed Computing

Distributed Computing covers the principles and practices of distributing computational tasks across multiple systems. Topics include distributed algorithms, communication protocols, fault tolerance, and distributed databases.

Back to Top

Comprehensive Viva

Comprehensive Viva is an oral examination where students demonstrate their overall understanding of the curriculum through answering questions posed by examiners.

Back to Top

Deep Learning

Deep Learning involves training complex neural networks to perform tasks like image and speech recognition, natural language processing, and autonomous driving. Topics include neural network architectures, training algorithms, and practical applications.

Back to Top

Programming Paradigms

Programming Paradigms explore different approaches to programming, including procedural, object-oriented, functional, and logical paradigms. Topics include language features, design patterns, and comparative analysis of paradigms.

Back to Top

Network Security Protocols

Network Security Protocols cover techniques to secure data transmission across networks. Topics include encryption, authentication, integrity verification, and the implementation of secure protocols like SSL/TLS, IPSec, and VPNs.

Back to Top

Soft Computing

Soft Computing uses approximation and probabilistic methods, like fuzzy logic, neural networks, and genetic algorithms, to solve complex problems. Topics include hybrid systems, adaptive algorithms, and real-world applications.

Back to Top

Embedded Systems

Embedded Systems involve designing and programming specialized computing systems embedded within larger devices. Topics include microcontroller programming, real-time operating systems, hardware-software integration, and application development.

Back to Top

Computer Vision

Computer Vision involves techniques to enable computers to interpret and understand visual information. Topics include image processing, feature extraction, object recognition, and applications in areas like surveillance, robotics, and medical imaging.

Back to Top

Fuzzy Set Theory and Applications

Fuzzy Set Theory and Applications study mathematical frameworks for dealing with uncertainty and vagueness. Topics include fuzzy logic, fuzzy inference systems, and applications in control systems, decision-making, and pattern recognition.

Back to Top


PROGRAM ELECTIVE IV

Mobile Computing

Mobile Computing covers the development and deployment of applications on mobile platforms. Topics include mobile hardware, operating systems, app development frameworks, network protocols, and security considerations.

Back to Top

Parallel Computing

Parallel Computing involves dividing tasks into sub-tasks that can be processed simultaneously. Topics include parallel algorithms, multi-threading, distributed computing, and the use of parallel architectures like GPUs and multi-core processors.

Back to Top

Data Compression Techniques

Data Compression Techniques study methods to reduce the size of data for storage or transmission. Topics include lossless and lossy compression, entropy encoding, transform coding, and practical applications in multimedia and communication systems.

Back to Top

Data Mining

Data Mining involves extracting valuable patterns and knowledge from large datasets. Topics include classification, clustering, association rule mining, anomaly detection, and the application of data mining techniques in various fields.

Back to Top

Client Server Architecture

Client Server Architecture describes a network design where client devices request services and resources from centralized servers. Topics include server configuration, client-server communication protocols, and the development of client-server applications.

Back to Top

Formal Methods and Tools

Formal Methods and Tools use mathematical techniques for specifying, developing, and verifying software and hardware systems. Topics include formal specification languages, model checking, theorem proving, and tools for formal verification.

Back to Top


PROGRAM ELECTIVE V

Bioinformatics

Bioinformatics applies computational techniques to analyze and interpret biological data. Topics include sequence alignment, structural bioinformatics, genomics, proteomics, and the development of bioinformatics tools and databases.

Back to Top

Image Processing

Image Processing involves techniques to enhance, analyze, and manipulate images. Topics include image enhancement, segmentation, feature extraction, and applications in areas like medical imaging, remote sensing, and computer vision.

Back to Top

Software Testing

Software Testing involves evaluating software applications to identify and fix defects. Topics include test planning, test case design, automated testing, performance testing, and the use of testing tools and frameworks.

Back to Top

Blockchain Technologies

Blockchain Technologies provide a decentralized, secure way to record transactions. Topics include blockchain architecture, consensus algorithms, smart contracts, and applications in cryptocurrencies, supply chains, and digital identity.

Back to Top

Computational Linguistics

Computational Linguistics combines computer science and linguistics to process and analyze natural language data. Topics include syntactic and semantic analysis, language modeling, machine translation, and natural language understanding.

Back to Top

Internet of Things

The Internet of Things (IoT) connects everyday devices to the internet, enabling them to send and receive data. Topics include IoT architectures, communication protocols, IoT security, and applications in smart homes, healthcare, and industrial automation.

Back to Top

High Performance Computing

High Performance Computing (HPC) involves using supercomputers and parallel processing techniques to solve complex computational problems. Topics include parallel programming, HPC architectures, performance optimization, and applications in scientific computing.

Back to Top

A Minor Course is a secondary field of study that complements the main degree program. Topics vary depending on the chosen minor, allowing students to gain additional expertise and broaden their knowledge.
